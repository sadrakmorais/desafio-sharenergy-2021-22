# ROTAS:

## Clientes:

```
GET /users : Retorna todos os usuários
```

```
GET /users/:userId : Retorna um usuário específico
```

```
POST /users : Cria um novo usuário
```

```
PUT /users/:userId : Atualiza um usuário
```

```
DELETE /users/:userId : Deleta um usuário
```
